Xtg: XML tools GUI

At the moment, Xtg requires the xmlstar toolkit ( included ), which it then uses to check a selected XML file for well-formedness and validity.

Discalimer:
I didn't write the xmlstar toolkit ( which does most of actual the work ), I only wrote the GUI.

Download latest version: Download Xtg v0.4

Changes from v0.3.1:

  • find the xmlstar binary even if it isn't in the current folder, but in the $PATH
  • fix files from folders containing spaces are shown encoded
  • clearer error messages: separate filename, line and column from message
  • get rid of hollow squares appearing in output
